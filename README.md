# 🎂 Age Calculator Website

## 📌 Project Overview
The **Age Calculator** is a responsive web application built using HTML, CSS, and JavaScript. It calculates the user's exact age based on their Date of Birth (DOB) and displays it in years, months, and days.

In addition to basic age calculation, this project includes interactive and modern UI features such as animations, gender selection, live countdown to the next birthday, and personalized user experience.

---

## ✨ Features

- 📅 Exact Age Calculation (Years, Months, Days)
- 👤 User Name Input
- ⚧ Gender Selection (Animated Radio Buttons)
- 🎂 Next Birthday Countdown (Live Timer)
- 💖 Animated Hearts Effect
- 🎉 Confetti Celebration Effect
- 🔊 Sound Effect on Result
- ✨ Typing Animation for Final Message
- 📱 Fully Responsive Design (Mobile Friendly)
- 🌈 Modern UI with Gradient and Glassmorphism

## 🚀 How It Works

1. User enters their **Name**
2. Selects **Gender**
3. Chooses **Date of Birth**
4. Clicks on **Calculate Button**
5. The system calculates:
   - Exact Age
   - Total days lived
   - Months & weeks
   - Next birthday countdown

Then it displays results with animations, effects, and a personalized message.
